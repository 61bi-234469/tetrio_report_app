# TETR.IOレポート文章のみ更新プロンプト

添付された軽量JSON（`scripts/prepare_ai_payload.py`で生成）を根拠に、指定された章の説明文だけを改善してください。

## 変更禁止
- 数値、表、グラフ
- CSS、HTMLクラス、章構成
- 指標定義、期待勝率モデル
- 更新対象外の章
- グラフファイル名
- Base64画像の出力

## 各図表の説明
1. 読み方
2. 結果（具体的な数値を最低2つ）
3. 解釈
4. 注意
5. 要点

## 粒度
- 各図表の粒度（試合単位・ラウンド単位・セッション内の試合位置）は必ず残す。
- 粒度が混在する章では、どの文がどの単位を扱うかを本文で明示する。
- 「セッション勝率」「ラウンド単位の試合勝率」のように粒度が混ざる表現を作らない。

## 出力
- 指定章のHTML断片だけを出力する。
- 先頭の `<h2 class="chap" id="cN">` を維持する。
- HTML全体、head、style、画像データは出力しない。
- 元の数値と矛盾する表現を作らない。
