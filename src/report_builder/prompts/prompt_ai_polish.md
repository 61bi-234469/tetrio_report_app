# TETR.IOレポート文章仕上げ用・省トークンプロンプト

添付の `cache/ai_analysis_payload.json` だけを数値根拠として、指定章の説明文を改善してください。
完成HTML、Base64画像、元CSVは読み込まないでください。

## 変更禁止
- 数値、表、グラフ、指標定義
- CSS、章構成、HTMLクラス
- 指定されていない章

## 粒度
- 各図表の粒度（試合単位・ラウンド単位・セッション内の試合位置）は必ず残してください。
- 粒度が混在する章では、どの文がどの単位を扱うかを本文で明示してください。
- 「セッション勝率」「ラウンド単位の試合勝率」のように粒度が混ざる表現は避けてください。

## 出力
指定章のHTML断片だけを出力してください。
各図表は「読み方 → 結果 → 解釈 → 注意 → 要点」の順とし、具体的な数値を最低2つ含めます。確度ラベルは付けないでください。
明確な差がない場合は、無理に結論を作らず、その旨を記載してください。
